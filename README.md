# Basile UI (React AI Chat Interface)

这是一个基于 React 和 Tailwind CSS 构建的现代化 AI 聊天界面组件，具有精美的磨砂玻璃（Glassmorphism）设计风格和流畅的动画效果。

该项目旨在提供一个功能完整的聊天 UI 前端，支持 Markdown 渲染、代码高亮、数学公式显示以及多模态交互模拟。

## ✨ 核心特性 (Features)

* **🎨 极致 UI 设计**:
    * **Glassmorphism**: 广泛使用的磨砂玻璃效果（`glass`, `glass-darker`），营造现代感。
    * **流畅动画**: 自定义 CSS 动画，包括背景光斑浮动 (`animate-blob`)、元素弹跳 (`animate-pop`) 和消息滑入效果。
    * **响应式布局**: 适配移动端和桌面端，带有可折叠的侧边栏。

* **📝 强大的内容渲染**:
    * **Markdown 支持**: 内置轻量级 Markdown 解析器，支持表格、列表、引用、粗斜体等。
    * **代码高亮**: 集成 **PrismJS**，支持多种语言（Python, JS, CSS 等）的语法高亮，并带有“复制”功能。
    * **数学公式**: 集成 **KaTeX**，支持行内 ($E=mc^2$) 和块级 ($$...$$) LaTeX 公式渲染。

* **🤖 交互功能**:
    * **流式响应模拟**: 模拟 AI 打字机效果，逐字显示回复。
    * **多模态输入**: 支持文本输入、语音输入（模拟）和图片上传（模拟）。
    * **聊天管理**: 支持新建聊天、重命名、固定（Pin）和删除历史记录。
    * **多模型切换**: 界面包含 Fast / Thinking / Pro 三种模型模式的选择菜单。

## 🛠️ 技术栈 (Tech Stack)

* **Frontend Framework**: React (Hooks: `useState`, `useEffect`, `useRef`)
* **Styling**: Tailwind CSS + Custom CSS (Animation Keyframes)
* **Icons**: `lucide-react`
* **External Libraries (CDN加载)**:
    * `PrismJS` (Syntax Highlighting)
    * `KaTeX` (Math Rendering)

## 🚀 快速开始 (Getting Started)

### 依赖安装
首先确保你安装了 React 环境和 Tailwind CSS，然后安装图标库：

```bash
npm install lucide-react
```

### 使用方法
该组件被设计为单文件组件（Single File Component）。

1.  将 `Basile UI Beta` 或 `Basile UI Stable` 的内容复制到你的 React 项目中的组件文件（例如 `App.js` 或 `ChatInterface.jsx`）。
2.  确保你的项目配置了 Tailwind CSS。
3.  **配置 API**:
    * 在代码中找到 `handleSend` 函数。
    * 填入你的 Google Gemini API Key：
        ```javascript
        // 查找此处并填入 Key
        const apiKey = "YOUR_API_KEY_HERE"; 
        ```
    * *注意：默认代码包含 API 调用逻辑，但 Key 为空。如果未配置 Key，代码会回退到模拟错误响应。*

## 🧩 自定义配置 (Customization)

* **修改主题色**: 可以在 `customStyles` 常量中修改动画的关键帧颜色（如 `bg-blue-300`, `bg-purple-300` 等 blob 颜色）。
* **添加更多语言高亮**: 在 `useExternalLib` 钩子中添加更多的 PrismJS 组件 CDN 链接。
